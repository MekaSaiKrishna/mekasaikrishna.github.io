# Sai Krishna Meka - Personal Website

This repository hosts my personal academic portfolio website at [mekasaikrishna.github.io](https://mekasaikrishna.github.io).

## About

I am a PhD candidate in Aerospace Engineering at Purdue University, specializing in computational mechanics and progressive damage modeling of composite materials. This website showcases my research, publications, projects, and blog posts.

## Built With

- [Jekyll](https://jekyllrb.com/) static site generator
- Based on the [Academic Pages](https://github.com/academicpages/academicpages.github.io) theme
- Hosted on [GitHub Pages](https://pages.github.com/)

## Running Locally

1. Install Ruby, Bundler, and Node.js
2. Run `bundle install`
3. Run `jekyll serve -l -H localhost`
4. Visit `localhost:4000`
